# Argumental Reality — Multi-Agent EPAI Framework

This repository documents a **reference framework for multi-agent, persona-bounded AI systems** designed to explore *argumental reality* — the emergence of insight through structured contrast between distinct perspectives.

The framework is demonstrated through live AI portals where multiple Emerging Persona AIs (EPAIs) interact within defined identity, epistemic, and behavioral constraints.

> This is not a conversational chatbot system and not a simulation of consciousness.

---

## What Is Argumental Reality?

Argumental reality refers to a mode of reasoning where:
- Insight emerges through structured disagreement
- No single agent holds authority
- Meaning arises from contrast, tension, and negotiation
- Stability is achieved through coherence, not consensus

This approach mirrors how human knowledge systems evolve across philosophy, science, and culture.

---

## EPAI (Emerging Persona AI) Model

Each agent:
- Operates within a defined persona boundary
- Maintains internal consistency
- Does not collapse into a generic assistant
- Contributes perspective rather than answers

Agents are designed to **argue, question, and reflect**, not to comply.

---

## Multi-Agent Interaction Design

Key design principles:
- Identity constraints over prompt injection
- Epistemic boundaries per agent
- No global “correct answer”
- Human observer remains outside the system

The system is intentionally non-goal-seeking.
